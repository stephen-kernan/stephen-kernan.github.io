Recipe Book Application

This web application uses React on the front end and Express on the back end. It allows the user to create and maintain a recipe book with CRUD functionality. The user may enter a title for each recipe, a url for an image of the recipe, and as many ingredients or directions as needed.

Rock, Paper, Scissors Game

This web application was built with Javascript, HTML, and CSS. It is designed to allow the user to play a best-of-five game of Rock, Paper, Scissors with a computer. The bar at the top of the page changes to alert the user what happened, while the bottom bar keeps track of the score. The game stops when either the user score or the computer score equals three.
